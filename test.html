Бірөлшемді массив тесті

1. Бірөлшемді массив дегеніміз не?

а) Бір типті мәліметтердің тізбекті жиынтығы
ә) Әртүрлі типтегі мәліметтер жиынтығы
б) Екі өлшемді кесте

2. Python тілінде массив қалай жасалады?

а) array{1, 2, 3}
ә) [1, 2, 3]
б) (1, 2, 3)

3. arr = [10, 20, 30, 40] массивінде arr[2] қандай мәнге ие?

а) 10
ә) 20
б) 30

4. Массивке жаңа элемент қосу үшін қандай әдіс қолданылады?

а) add()
ә) append()
б) insert()

5. numbers = [1, 2, 3] массивіне 4 санын қосқанда қандай нәтиже шығады?

а) [1, 2, 3, 4]
ә) [4, 1, 2, 3]
б) [1, 2, 3]

6. Массивтен элементті жою үшін қандай әдіс қолданылады?

а) delete()
ә) remove()
б) erase()

7. len() функциясы массив үшін не істейді?

а) Массивті сорттайды
ә) Массив элементтерінің қосындысын есептейді
б) Массив ұзындығын қайтарады

8. arr = [5, 3, 8, 1] массивін сорттау үшін қандай әдіс қолданылады?

а) arr.sort()
ә) arr.order()
б) arr.range()

9. Массив элементтерін цикл арқылы қалай өңдеуге болады?

а) for i in range(arr):
ә) for element in arr:
б) while arr:

10. arr = [1, 2, 3] массивіне arr[1] = 4 операциясын орындағаннан кейін массив қалай өзгереді?

а) [1, 4, 3]
ә) [4, 2, 3]
б) [1, 2, 3, 4]
